# Liquid Glass Template

A modern, interactive glassmorphism UI component with liquid distortion effects and drag functionality.

## Features

- **Glassmorphism Effect**: Translucent glass-like appearance with backdrop blur
- **Liquid Distortion**: Dynamic SVG filter creating realistic glass distortion
- **Specular Highlights**: Light reflection effects that enhance visual depth
- **Draggable Interface**: Click and drag the glass card anywhere on the screen
- **Responsive Design**: Centered layout that works on different screen sizes

## Visual Effects

The template combines multiple layered effects:
- Backdrop blur for the glass effect
- SVG turbulence and displacement mapping for liquid distortion
- Specular lighting for realistic light reflections
- Gradient icon with smooth shadows
- Semi-transparent tinting and shine overlays

## Usage

Simply open `index.html` in a web browser. Click and drag the glass card to move it around the screen.

## Customization

You can customize the following:
- Background image (currently uses a Pexels photo)
- Glass card dimensions and padding
- Blur intensity (`backdrop-filter: blur(3px)`)
- Distortion parameters in the SVG filter
- Color scheme and gradients
- Content inside the card

## Technologies

- Pure HTML/CSS/JavaScript (no dependencies)
- SVG filters for advanced visual effects
- CSS glassmorphism techniques
